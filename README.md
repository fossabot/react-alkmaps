# react-alkmaps

## Install

```bash
npm install --save react-alkmaps
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'react-alkmaps'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
